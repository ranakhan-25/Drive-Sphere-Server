# ⚙️ Car Rental Backend API

This is the backend service for the **Car Rental Booking App**.  
It provides RESTful APIs for car listings, user authentication, and booking management.  
Authentication is handled using **Better-Auth** with **JSON Web Tokens (JWT)** for secure session handling.

---

## 🚀 Tech Stack
- **Node.js + Express.js** — API server
- **MongoDB + Mongoose** — Database
- **Better-Auth** — Authentication
- **JWT (JSON Web Token)** — Secure session & authorization
- **Cors + dotenv** — Config & security
